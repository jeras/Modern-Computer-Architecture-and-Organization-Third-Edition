# Chapter 19, Exercise 1

 Install the Qiskit quantum processor software development framework by following the instructions at https://quantum.cloud.ibm.com/docs/en/guides/install-qiskit.

# Answer
Install Qiskit and related dependencies with the following command:
```
pip install qiskit qiskit-aer qiskit-ibm-runtime matplotlib
```
